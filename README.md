# Client Generator

VS Code / Cursor extension for generating Python API clients from JSON collections (Postman or OpenAPI).

## Установка из VSIX

Готовый пакет: [`vscode-extension/postman_client_generator.vsix`](vscode-extension/postman_client_generator.vsix)

```bash
cursor --install-extension vscode-extension/postman_client_generator.vsix
# или
code --install-extension vscode-extension/postman_client_generator.vsix
```

После установки в Activity Bar появится **Client Generator**. Перезапустите окно, если иконки ещё нет.
